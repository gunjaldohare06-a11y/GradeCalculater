# 🎓 **Student Grade AI Calculator**

A modern and interactive **Student Grade Calculator** built using **Python & Streamlit**. This application helps students calculate their total marks, percentage, and grade instantly through a beautiful and user-friendly interface.

---

# 🚀 **Features**

✔️ Calculate Total Marks Automatically  
✔️ Calculate Percentage Instantly  
✔️ Automatic Grade Generation  
✔️ Modern Dark UI Design  
✔️ Fast and Accurate Results  
✔️ User-Friendly Interface  
✔️ Real-Time Calculations  
✔️ Responsive Layout

# ⚙️ **Installation**

### **1. Clone the Repository**

```bash
git clone https://github.com/your-username/GradeCalculator.git
```

### **2. Open Project Folder**

```bash
cd GradeCalculator
```

### **3. Install Required Packages**

```bash
pip install -r requirements.txt
```

### **4. Run the Application**

```bash
streamlit run app.py
```
